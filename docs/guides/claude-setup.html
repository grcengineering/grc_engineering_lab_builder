Follow these steps to use your generated prompts with Claude Projects

Step 1: Create a New Project

  1. Go to claude.ai
  2. Click on "Projects" in the left sidebar
  3. Click "Create Project"
  4. Name it something like "GRC Learning Lab"

Step 2: Add Your System Prompt

  1. In your new project, click "Add content"
  2. Select "Add text"
  3. Paste your System Prompt (the first prompt from the generator)
  4. Click "Add" or "Save"

Step 3: Start a Chat

  1. Click "Start chat" in your project
  2. Paste your User Prompt (the second prompt from the generator)
  3. Press Enter
  4. Claude will generate your personalized learning lab!
üí° Pro Tip: You can reuse the same project for multiple labs! Just start a new chat and paste a different User Prompt. Your GRC profile (System Prompt) stays loaded in the project.
